# 🥗 Nutrition Bot — Дэйвид

Персональный нутрициолог в Telegram с распознаванием еды по фото.

## Возможности
- 📸 Распознавание еды по фото → автоматический подсчёт КБЖУ
- 📊 Дневник питания с прогресс-барами
- 🍽 Меню на неделю под лего-заготовки
- 🛒 Список покупок
- 💬 Вопросы по питанию и тренировкам

## Deploy на Railway

### 1. Переменные окружения (Environment Variables)
В Railway добавь:
```
TELEGRAM_TOKEN=твой_токен_от_BotFather
ANTHROPIC_API_KEY=твой_ключ_sk-ant-...
```

### 2. Deploy
- Загрузи файлы в GitHub репозиторий
- Подключи репозиторий в Railway
- Railway автоматически запустит бота

## Команды бота
- `/start` — главное меню
- `/diary` — дневник за сегодня  
- `/reset` — сбросить дневник
